# SignalMatch Role Pack

This is the paid download for SignalMatch.

It is designed for candidates applying to:

- product
- operations
- analyst
- growth
- customer success

It is built around one rule:

Do not invent proof.

Use these files to sharpen what is already true, make weak experience more
legible, and prepare better interviews faster.

## Files

- `01-role-targeting-sheet.csv`
- `02-proof-harvester.md`
- `03-bullet-rewrite-rubric.md`
- `04-interview-bank.md`
- `05-7-day-application-sprint.md`
- `06-linkedin-announcement-template.md`

## Recommended Use

1. Pick one target role.
2. Fill out the targeting sheet.
3. Use the proof harvester before rewriting bullets.
4. Rewrite only the bullets you can defend.
5. Use the interview bank to rehearse stories out loud.
6. Follow the 7-day sprint when you are under deadline pressure.
